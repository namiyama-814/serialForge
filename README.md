# serialForge
シリアルナンバーを発行するシステムです。<br>
MIT Licenseの下で、自由に利用・改変・再配布が可能です。

## リンク
### シリアルナンバーを発行する
```
serialforge.onrender.com/make
```
### シリアルナンバーを利用する
```
serialforge.onrender.com/use
```
## 注意点
<ul>
<li>このシステムは簡易的な設計であり、セキュリティ上の脆弱性が存在します。</li>
<li>本格的な業務や重要データの運用には使用しないでください。</li>
<li>データの永続保存を保証していません。必要なデータは別途バックアップを取ってください。</li>
</ul>
